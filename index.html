Timeline

誕生

長野で冬季オリンピックがあった年に生まれ、父の外見と母の性格が合わさった男の子として成長する。 父も母もパソコンに詳しく、パソコンと技術書に囲まれた環境で育った。

趣味は囲碁

6歳のときに祖父から囲碁を教わる。 負けず嫌いだったのもあってよく勉強し、父や祖父に勝てるようになるまでになった。 小学6年のときには長野市の小学生囲碁大会で3位を取った。 正式ではないが現在は初段くらいはある。 余談だが、囲碁以外にも将棋・チェス・象棋のルールも把握している。 友達と将棋やチェスで遊んだときは割と強い方であった。

パソコン検定3級を取得

中学の部活で「技術部」という名のパソコンで遊んでいる部活があり、 そこでパソコン検定という、タイピングやネットワーク基礎や Word, Excel, PowerPointの使い方などのパソコンを使って行う試験があり、その3級を取得している。

長野高専の情報科に入学

高専一年目の授業で Processing を扱って以来プログラミングは唱えると動く 「リアル魔法」だと考えるようになり、プログラミングの素晴らしさに気づく。 この後、C, Java, JavaScript, PHP に手を伸ばす。 しかしWeb関連の勉強（PHPあたり）に入るとサーバの話も混じり、勉強しないといけないことが山ほどあると知る。

Ruby信者となる

このころ、Rubyのパパ（まつもとゆきひろ）さんが長野高専の講演会に来ると聞き、Rubyの勉強を始める。 講演会の後、自分も人の役に立つコードを書きたいと思うようになる。 この後、RubyのメタプログラミングでDSLを書いたり、RakeでLaTeXのビルドの自動化をしたり、 testunitでテスト駆動開発をしたりして遊ぶ。

学生会長賞に「Ecoris」が表彰される

学園祭のクラス企画でEcoris（3Dテトリス + エコ）という LeapMotion で行う3Dテトリス のゲームのプログラム（C#）部分を担当し「学生会長賞」を勝ち取った。 使用環境は Unity + C# + LeapMotion


Skills

プログラミング言語は多くの言語から影響されて作られた経緯があるため、幾つかの言語を同時に勉強するのが効率がいいかなと思っている。 もちろん勉強した全ての言語を使いこなせるという訳ではないが、勉強した中で特に好きな言語（~= 得意な言語）について紹介する。

Languages

Ruby

純粋なオブジェクト指向・美しい構文・メタプログラミングの柔軟性 の三拍子が揃っているため、 プロンプト上においては最も効率的にプログラムが書ける言語の1つだと思っている。 Perlの正規表現と同等の機能を持つ点もポイント。

Elixir

Ruby風な構文で書ける関数型言語。ErlangのVM上で動く。 関数型言語の考え方と、Erlangのシステムと、Rubyのように扱えるオブジェクトが合わさった素晴らしい言語。 macroによるメタプログラミングも強力。

Python

「やりかたは明確な一つの方法にすべき」という考え方が強い。 コード自体に暗黙的な部分が少なく、誰が書いても読めるコードになるため、複数人で共同開発をするときとかに重宝する。 コンパイル & 実行速度も早い。

CoffeeScript

JavaScriptをRubyやPython風な構文で書ける言語。 Rubyの良いところとPythonの良いところ（特にリスト内包記法など）を合わせた言語。 変換後はコメントが消えてしまうが、比較的読みやすいコードが生成される。

Shell

様々な環境でデフォルトで用意されており、POSIXによって仕様変更がほとんどないので、長期の運用に向いている言語。 awk, sed, make (Makefile) はGNU拡張を除いて使える（FreeBSD派）。 目標は「コピー一発、デプロイ完了」

Environment

Atom

Github製のエディタ。スニペットやパッケージの追加が楽なのでメインのiMacで使用している。

Git

バージョン管理システム。Ubuntuではコマンドから、MacOSではSourceTreeを通して使用している。

GitHub

Gitのバージョン管理を複数人で共有するためのWebサービス。 クラス企画の作業はGithubを通して行っている。

Sublime

どこにでも持ち運び可能な軽量エディタ。乏しい環境下でもコーディングのパフォーマンスを落とさずに済む。

Heroku

RubyやNode.jsなどが動くサーバを無料で提供しているサービス。 不便な部分もあるが、とりあえずこれで我慢。
